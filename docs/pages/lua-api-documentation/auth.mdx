## auth:get_servers()
```lua
table<number, Server> auth:get_servers()
```

### Server type
```lua
--[[
	bOnline
	bUpg
	iChat
	iCount
	iLevel
	iMax
	iPort
	sIP
	sLang
	sName
]]
```

### example
```lua
-- get the list of online server names
for k, v in ipairs(auth:get_servers()) do
	if v.sOnline then
		print(v.sName)
	end
end
```
