### level <Badge text="Getter" size="small" />

### maxHp <Badge text="Getter" size="small" />

### maxMp <Badge text="Getter" size="small" />

### mp <Badge text="Getter" size="small" />

### pad <Badge text="Getter" size="small" />

Whether the player has the specified aura. If a value is provided,

| value | `number` | The value to check. |
