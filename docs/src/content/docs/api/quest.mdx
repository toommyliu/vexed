### cmd.abandon_quest
Abandons a quest.

| Parameter | Type | Description |
|-----------|------|-------------|
| questId | `number` | The ID of the quest. |


### cmd.accept_quest
Accepts a quest.

| Parameter | Type | Description |
|-----------|------|-------------|
| questId | `number` | The ID of the quest. |


### cmd.complete_quest
Completes a quest.

| Parameter | Type | Description |
|-----------|------|-------------|
| questId | `number` | The ID of the quest. |
| itemId | `number?` | Optional item reward ID to select. |


### cmd.register_quest
Registers one or more quests, which automatically handles accepting and completing them.

| Parameter | Type | Description |
|-----------|------|-------------|
| questIds | `number` \| `[number, number]` \| `(number` \| `[number, number])[]` | Quest ID, array of quest IDs, or array containing IDs and [questId, itemId] tuples. |
| itemId | `number?` | Optional item ID when passing a single quest ID. |

<Code code={`// Single quest
cmd.register_quest(1234);

// Single quest with item ID
cmd.register_quest(1234, 5678);

// Multiple quests
cmd.register_quest([1234, 5678, 9012]);

// Multiple quests, some with item IDs
cmd.register_quest([1234, [5678, 9999], 9012]); `} lang="js" />


### cmd.unregister_quest
Unregisters one or more quests.

| Parameter | Type | Description |
|-----------|------|-------------|
| questIds | `number` \| `number[]` | The ID or array of IDs of the quests. |
