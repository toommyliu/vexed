### cmd.buy_item
Buys an item from the shop.

| Parameter | Type | Description |
|-----------|------|-------------|
| shopId | `number` | The shop id. |
| item | `string` \| `number` | The name or item id of the item to buy. |
| quantity | `number` | The quantity of the item to buy. |


### cmd.deposit
Puts an item into the bank.

| Parameter | Type | Description |
|-----------|------|-------------|
| item | `string` \| `number` \| `(string` \| `number)[]` | The item to deposit. If provided an array, each item will be deposited. |

### cmd.equip_item_by_enhancement
Equips an item using its enhancement name. Supports colloquial variants of the enhancement name.

If results are ambiguous, the first matching item will be used. Use itemType to narrow the result.

| Parameter | Type | Description |
|-----------|------|-------------|
| enhancementName | `string` | The name of the enhancement. |
| itemType | `string?` | The type of item to equip. Can be "weapon", "helm", or "cape". |

### cmd.pickup
Picks up an item from the drop list.

| Parameter | Type | Description |
|-----------|------|-------------|
| item | `string` \| `number` | The name or item id of the item to pick up. |


### cmd.register_boost
Registers a consumable boost to use when available.

| Parameter | Type | Description |
|-----------|------|-------------|
| item | `string` | The name of the item boost to use. |

### cmd.reject
Rejects an item from the drop list.

| Parameter | Type | Description |
|-----------|------|-------------|
| item | `string` \| `number` | The name or item id of the item to reject. |


### cmd.sell_item
Sells an item.

| Parameter | Type | Description |
|-----------|------|-------------|
| item | `string` | The name of the item to sell. |


### cmd.swap
Swaps an item from the bank with an item from the inventory.

| Parameter | Type | Description |
|-----------|------|-------------|
| bankItem | `string` \| `number` | The name or item id of the item to swap from the bank. |
| invItem | `string` \| `number` | The name or item id of the item to swap from the inventory. |

| Parameter | Type | Description |
|-----------|------|-------------|
| item | `string` | The name of the item boost to unregister. |


### cmd.unregister_drop

| Parameter | Type | Description |
|-----------|------|-------------|
| items | `string` \| `string[]` | The name or item id of the item(s) to unregister. |


### cmd.withdraw
