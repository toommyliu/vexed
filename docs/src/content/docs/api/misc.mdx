Buys the specified quantity of Scroll of Life Steal.

| Parameter | Type | Description |
|-----------|------|-------------|
| qty | `number` | The quantity to buy. |


### cmd.delay

|-----------|------|-------------|
| ms | `number` | The delay in milliseconds. |
