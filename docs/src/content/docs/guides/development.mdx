## Prerequisities

1. [Node.js / npm](https://nodejs.org/en) AND [yarn classic](https://classic.yarnpkg.com/en/docs/install#mac-stable)
2. [git](https://git-scm.com/downloads)

Optionally:

1. [OpenJDK](https://openjdk.org/)
2. [Flash SDK](https://moonshine-ide.com/download-sdk-installer/)

<Aside>
  Any modern version of Node.js should work fine. As long as its LTS or higher,
  you'll probably be ok.
</Aside>

## First steps

1. Clone the repository (`git clone`)
2. Install dependencies (`yarn install`)
