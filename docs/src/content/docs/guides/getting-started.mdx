## Downloading

Download the latest version (tagged `nightly`) from the [releases page](https://github.com/toommyliu/vexed/releases/tag/nightly).

Select the correct file for your platform.

## Running the app

### macOS

When running the app for the first time, you might see a warning along the lines **Apple cannot check it for malicious software**. To solve this, click **OK** then open Finder, find the app, and hold **CTRL** and right click the icon and press **Open**.

It might also tell you to install Rosetta, which is required to support x86_64 apps on Apple Silicon machines.

### Windows

No further steps are required.

## Updates

All future updates must be downloaded manually.

We may serve notifications for available updates in the future. See [#47](https://github.com/toommyliu/vexed/issues/47).
