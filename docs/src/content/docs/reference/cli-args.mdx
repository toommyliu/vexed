## Game Options

Launch the game with the following options:

### --game, -g

```bash
--game, -g
```

Launches a game window.

### Additional Game Options

The following options are available when using the game option:

#### --username

```bash
--username="username"
```

Specifies the username for authentication.

#### --password

```bash
--password="password"
```

Specifies the password for authentication.

#### --server

```bash
--server="server"
```

Specifies the server to connect to.

#### --scriptPath

```bash
--scriptPath="/path/to/script.js"
```

Specifies the path to a script file to execute.

## Manager Options

### --manager, -m

```bash
--manager, -m
```

Launches the account manager.
