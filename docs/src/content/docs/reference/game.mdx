The game client looks like the typical AQW third-party client.

You can always open another game client window by clicking the tray icon and selecting **Open Game**.

## Top Navigation Bar

The top navigation bar provides quick access to core features. From left to right:

### Windows Dropdown

Opens a mega menu containing shortcuts to all application windows:

- **Environment** - Manage registered quests, drops, and boosts
- **Logs** - View application log messages
- **Hotkeys** - Configure keyboard shortcuts
- **Fast Travels** - Quick travel to popular areas
- **Loader/Grabber** - Load and grab game data
- **Follower** - Follow players around the map
- **Packet Logger** - Log outgoing packets
- **Packet Spammer** - Send packets to the server

### Scripts Dropdown

- **Load Script** - Opens a file dialog to load a script file
- **Show/Hide Overlay** - Toggles the command overlay visibility
- **Dev Tools** - Opens Chrome DevTools for debugging

### Options Button

Opens the options panel for quick access to game modifications and settings.

### Run/Stop Button

Starts or stops the currently loaded script. Only enabled when a script is loaded.

<Aside>
  Scripts can only begin executing when the player is in a
  [ready](/api-legacy/player#isready) state.
</Aside>

### Auto Checkbox

Enables auto-attack mode. When enabled, the player will automatically attack monsters and use skills based on configured skill sets.

### Cell/Pad Selectors

Two dropdown menus for selecting the target cell and pad. Valid pads for the current cell are highlighted.

### Jump Button

Jumps to the selected cell and pad combination.

### Bank Button

Opens or closes the bank interface.

## Menu Bar

Additional features are available in the application menu bar.

### Command Palette

Open the command palette with `Cmd+K` (macOS) or `Ctrl+K` (Windows) to quickly access commands and toggle options. Available under **View → Command Palette**.

### Appearance

Change the application theme. Available under **View → Appearance**:

- **Light** - Light color scheme
- **Dark** - Dark color scheme  
- **System** - Follow system preference

## Options

<Aside type="tip">Settings are a two-way binding between the ui and api.</Aside>

### Infinite Range

Allows the player to attack from any distance.

### Provoke Map

Tags all monsters in the map.

### Provoke Cell

Tags all monsters in the cell.

### Enemy Magnet

Sets the position of the current target to the player's position.

### Lag Killer

Mostly disables rendering.

### Hide Players

Hide all other players.

### Skip Cutscenes

Skips cutscenes when possible.

### Disable FX

If checked, mostly disables animations and effects.

### Disable Collisions

If checked, disables the map's collision mechanics.

### Walk Speed

The player's walk speed.

### FPS

The target client FPS.
