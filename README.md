# vexed

Crossplatform AQW "commands"-based Scripting Client

## Alternatives

Windows users who prefer a more mature tool or larger communities may want to explore [Skua](https://github.com/auqw/Skua) or [Grimlite-Li](https://github.com/Froztt13/Grimlite-Li). This project builds on them and their predecessors, and incorporates some of their code to replicate key functions.
